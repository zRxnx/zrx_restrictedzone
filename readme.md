# zRxnx's Advanced Restricted Zone System

An advanced restricted zone system for FiveM

**https://docs.zrxnx.at**

**https://discord.gg/mcN25FJ33K**

## About

### Features

- Template System
- Multiple zones support
- Notifications
- Custom radius
- Speedlimit
- Automatic zone removal
- Remove NPC cars
- Sounds
- Remove existing blips
- ox_lib log
- Update checker
- Optimized
- Synced
- 0.0 ms on idle
- 0.0 ms while in use

## Requirements

- ox_lib (latest)

## Installation

- Download script from Releases
- Put script in your `resources` directory
- Add `ensure zrx_restrictedzone` in your `server.cfg`
